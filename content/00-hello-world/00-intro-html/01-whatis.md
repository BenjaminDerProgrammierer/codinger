HTML is the base lanuguage powering the web. The first version was created in 1989 by Tim Berners-Lee, a contractor at CERN. HTML defines the structure of a web page by using so-called *Tags*, which look like this: `<p>`. An end tag has a `/` character like this: `</p>`. Together, they form an element: `<p>Some Text</p>` The current version of HTML is HTML5. It was published in January 2008, and is the recommended default since 2014.

Here you can see a classic HTML "Hello World" page:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Hello World</title>
  </head>
  <body>
    <main>
        <p>Hello world!</p>
    </main>
  </body>
</html>
```
